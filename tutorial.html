Langkah 1: Membuat Repository

Buka github.com dan login menggunakan akun GitHub. Klik tombol New Repository di dashboard.

Masukkan nama repository (misalnya: company-profile), aktifkan opsi Public, dan centang Initialize this repository with a README agar langsung bisa digunakan.

Langkah 2: Upload File Website

Setelah repository dibuat, buka tab Code, lalu klik tombol Add file â†’ Upload files.

Unggah file index.html, style.css, script.js, serta semua folder yang diperlukan seperti assets atau pages. Setelah selesai, klik Commit changes.

Langkah 3: Aktifkan GitHub Pages

Buka tab Settings di repository tersebut. Scroll ke bagian Pages di sidebar kiri (atau klik tab "Pages" langsung).

Pada bagian Source, pilih opsi Deploy from a branch. Pilih branch: main dan folder / (root). Klik tombol Save.

Langkah 4: Akses Website

Setelah beberapa detik, GitHub akan menampilkan link website aktif. Link biasanya berbentuk:
https://username.github.io/nama-repository/

Klik link tersebut untuk melihat website online secara gratis.

Tips Tambahan

  • Gunakan nama file utama index.html agar otomatis ditampilkan sebagai halaman depan.
  • Struktur folder harus rapi dan saling terhubung melalui path relatif.
  • Untuk update konten, ulangi proses commit atau upload ulang file yang diperbarui.
